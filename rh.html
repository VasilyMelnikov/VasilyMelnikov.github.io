The Riemann hypothesis for function fields
Let \(K\) be a global field. Then \(K\) is either a finite extension of \(\mathbf{Q}\), or the function field of a smooth, projective curve \(C\) over \(\mathbb{F}_{q}\). I will assume the latter. The Riemann hypothesis for the function field of \(C\) is then the inequality $$\vert{N_{r}-(1+q^{r})}\vert\leq 2g\sqrt{q^{r}}$$ where \(N_{r}\) is the number of \(\mathbb{F}_{q^{r}}\)-rational points. This is different from the usual formulation, in which the Riemann hypothesis is viewed as a statement about the zeta function \(\zeta_{C}\) associated to \(C\).
